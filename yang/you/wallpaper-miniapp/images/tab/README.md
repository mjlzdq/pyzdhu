# 🖼️ 高清壁纸小程序

一款基于微信小程序 + 云开发的壁纸/头像应用，支持浏览、搜索、收藏、下载壁纸，通过激励视频广告变现。

## 📱 功能

| 功能 | 说明 |
|------|------|
| 🏠 首页 | 瀑布流展示壁纸，分类快捷切换 |
| 📂 分类 | 12个分类网格浏览，精准查找 |
| 🔍 搜索 | 关键词搜索，搜索历史，热门推荐 |
| 📄 详情 | 大图预览，标签信息，分享好友 |
| ❤️ 收藏 | 收藏喜欢的壁纸，本地存储 |
| 🕐 历史 | 浏览记录，支持清空 |
| 📥 下载 | 激励视频广告后下载高清原图 |
| 💰 变现 | Banner广告 + 激励视频 + 插屏广告 |

## 🚀 快速开始

### 1. 注册小程序账号
前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序账号（个人即可）

### 2. 开通云开发
- 在微信开发者工具中打开项目
- 点击「云开发」按钮开通
- 创建环境（选择基础版 19.9元/月 或免费额度）

### 3. 修改配置
- `app.js` 中将 `env: 'your-env-id'` 改为你的云开发环境 ID
- `project.config.json` 中将 `appid` 改为你的小程序 AppID

### 4. 初始化数据库
1. 在云开发控制台创建集合：`wallpapers`
2. 上传并运行 `cloudfunctions/initDatabase` 云函数
3. 或将你自己的壁纸数据导入 `wallpapers` 集合

### 5. 配置广告（上线前）
- 在微信公众平台「流量主」开通广告
- 将广告单元 ID 替换到 `app.js` 和 `pages/detail/detail.js` 中

### 6. 上传 tab 图标
在 `images/tab/` 目录下放入 8 个 tab 图标（PNG，81x81px）

### 7. 运行项目
用微信开发者工具打开项目目录，编译预览即可。

## 📊 数据库设计

### wallpapers 集合

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | String | 自动生成 |
| url | String | 壁纸原图URL |
| thumbUrl | String | 缩略图URL |
| width | Number | 图片宽度 |
| height | Number | 图片高度 |
| category | String | 分类（风景/动漫/简约等） |
| tags | Array | 标签 |
| likes | Number | 点赞数 |
| views | Number | 浏览数 |
| downloads | Number | 下载数 |
| createTime | Number | 创建时间戳 |

## ⚠️ 注意事项

1. **图片版权**：务必使用 AI 生成或免版权图片，避免侵权投诉
2. **广告单元 ID**：开发环境用测试 ID，上线前替换为正式 ID
3. **类目选择**：个人主体选「工具-图片处理」
4. **流量主门槛**：累计 UV ≥ 500 才能开通
5. **云开发费用**：起步阶段免费额度够用，日活大了再升级
